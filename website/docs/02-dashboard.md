# Dashboard & Übersicht

Das Dashboard ist Ihr zentraler Einstiegspunkt in Rechnix. Hier erhalten Sie sofort einen Überblick über die wichtigsten Kennzahlen.

![Dashboard](pathname:///img/docs/dashboard.png)

## Elemente des Dashboards

1.  **Sidebar (Seitenleiste)**:
    *   Links finden Sie die Navigation zu allen Programmbereichen: Dashboard, Rechnungen, Angebote, Lieferscheine, Kunden, Produkte, Statistiken, Ausgaben und Einstellungen.
    *   Die Sidebar ist immer sichtbar, damit Sie schnell navigieren können.

2.  **Statistik-Karten**:
    *   Zeigen den aktuellen Umsatz, offene Rechnungen und weitere Schnellstatistiken.

3.  **Schnellaktionen**:
    *   Buttons, um direkt eine neue Rechnung oder einen neuen Kunden anzulegen.

## Navigation

*   **Rechnungen**: Übersicht aller erstellten Rechnungen.
*   **Kunden**: Verwaltung Ihres Kundenstamms.
*   **Produkte**: Inventar Ihrer Produkte und Dienstleistungen.
*   **Ausgaben**: Erfassung von Belegen und Kosten.
*   **Statistiken**: Detaillierte Finanzanalysen.
