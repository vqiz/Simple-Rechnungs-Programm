## Neues Produkt anlegen

Klicken Sie oben rechts auf **"Produkt erstellen"**. Ein Formular öffnet sich:

import CreateProduktMock from '@site/src/components/DocsMock/CreateProduktMock';

<CreateProduktMock />

### Felder ausfüllen:

1. **Kategorie** auswählen (z.B. "Dienstleistung", "Hardware")
   - *Tipp: Wenn Sie noch keine Kategorie haben, können Sie erst eine erstellen*
2. **Produktname** eingeben
   - *Hinweis: Zeitbasierte Produkte müssen "stunde" im Namen enthalten*
3. **Netto-Preis** festlegen
4. **Mehrwertsteuersatz** auswählen:
   - 19% (Standard)
   - 7% (Ermäßigt)
   - 0% (Steuerfrei)
5. Auf **"Hinzufügen"** klicken

## Produkte bearbeiten

In der Tabelle finden Sie rechts neben jedem Produkt ein Menü (drei Punkte). Hier können Sie:
- Den **Titel** bearbeiten
- Den **Preis** anpassen
- Den **Steuersatz** ändern
- Das Produkt **löschen**
