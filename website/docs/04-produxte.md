## Produktübersicht

Die Produkte werden in einer übersichtlichen Tabelle dargestellt. Sie können nach Produktnamen oder Kategorien suchen.

## Neues Produkt anlegen

1.  Klicken Sie oben rechts auf **"Produkt erstellen"**.
2.  Wählen Sie eine **Kategorie** aus (z.B. "Dienstleistung", "Hardware").
    *   *Tipp: Wenn Sie noch keine Kategorie haben, können Sie erst eine erstellen.*
3.  Geben Sie den **Produktnamen** ein.
4.  Legen Sie den **Netto-Preis** fest.
5.  Der Steuersatz wird standardmäßig auf 19% gesetzt, kann aber angepasst werden.
6.  Klicken Sie auf **"Hinzufügen"**.

## Produkte bearbeiten

In der Tabelle finden Sie rechts neben jedem Produkt ein Menü (drei Punkte). Hier können Sie:
*   Den **Titel** bearbeiten.
*   Den **Preis** anpassen.
*   Den **Steuersatz** ändern.
*   Das Produkt **löschen**.
