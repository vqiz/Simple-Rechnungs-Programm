# Rechnungen erstellen

Das Herzstück von Rechnix ist die einfache Erstellung von Rechnungen.

![Rechnungsübersicht](pathname:///img/docs/rechnungen_list.png)

## Neue Rechnung erstellen

1.  Gehen Sie im Menü auf **"Rechnungen"**.
2.  Klicken Sie auf **"Rechnung erstellen"**.
3.  **Kunde wählen**:
    *   Wählen Sie einen bestehenden Kunden aus der Liste oder erstellen Sie einen neuen.
4.  **Rechnungsdaten**:
    *   Das Datum wird automatisch auf "Heute" gesetzt, kann aber geändert werden.
    *   Geben Sie bei Bedarf eine Leistungsbeschreibung an.
5.  **Positionen hinzufügen**:
    *   Klicken Sie auf **"Position hinzufügen"**.
    *   Wählen Sie ein Produkt aus Ihrem Inventar oder geben Sie einen freien Text ein.
    *   Passen Sie Menge und Preis an.
6.  **Speichern & Vorschau**:
    *   Die Rechnung wird automatisch gespeichert.
    *   Klicken Sie auf das "Auge"-Symbol, um die Vorschau zu öffnen.

## Rechnungsvorschau & Aktionen

In der Vorschau sehen Sie die Rechnung genau so, wie sie als PDF exportiert wird.

*   **PDF Export**: Klicken Sie auf das PDF-Symbol, um die Rechnung zu speichern.
*   **Drucken**: Drucken Sie die Rechnung direkt aus der Anwendung.
*   **E-Rechnung**: Exportieren Sie eine XML-Datei für den elektronischen Rechnungsaustausch (XRechnung Format).
*   **Per E-Mail senden**: Nutzen Sie die E-Mail-Funktion, um die Rechnung direkt zu versenden.

## Rechnungsstatus

Rechnungen haben verschiedene Status (z.B. Offen, Bezahlt). Sie können den Status in der Listenansicht ändern, um den Überblick über offene Beträge zu behalten.
